Fairytale Maze Game
A fun, interactive web-based maze game with a fairytale theme, built using pure frontend technologies.

🌟 Features
Interactive Gameplay: Navigate through a challenging maze to reach the finish line.
Responsive Design: Styled with CSS to provide an immersive fairytale atmosphere.
Dynamic Logic: Powered by JavaScript to handle character movement and win/loss conditions.
Lightweight: No external frameworks or libraries required—just pure HTML, CSS, and JS.

🛠️ Tech Stack
HTML
CSS3
JavaScript 

🎮 How to Play
1. Clone or Download this repository to your computer.
2. Locate the index.html file in the main folder.
3. Double-click index.html to open it in any modern web browser (Chrome, Firefox, Edge, etc.).
4. Use your keyboard arrow keys (or on-screen controls, if applicable) to guide your character through the maze!
